# OnlinepA - Landing Page de Alta Conversão

## 🛠 Stack Tecnológica
- **Framework:** Next.js (App Router)
- **Estilização:** Tailwind CSS v4
- **Backend/Auth:** Supabase
- **Pagamentos:** Mercado Pago
- **Deploy:** Vercel

## 👤 Perfil do Desenvolvedor & Regras
- **Cláudio:** TI desde 1995. Abordagem pragmática e técnica.
- **Estilo:** Solitude (foco em eficiência, sem sugestões sociais).
- **Design:** Minimalista, alta conversão, padrão Anthropic (Dobra 1-5).
